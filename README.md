1. 프로젝트 명과 프로젝트 기능

-프로젝트 명: 스도쿠 게임

-프로젝트 기능:
게임 시작 및 난이도 선택->게임 진행->1.게임 완료 여부 확인 및 추가 옵션 제공 ->새 게임을 시작하거나 프로그램을 종료할 수 있는 선택지를 제공
                                  2.게임 도중 포기(종료) 옵션-> 포기 시 답안 보기 선택->1.답안을 보겠다 선택시 답안을 보여주고 종료
                                                                                    2.답안을 안보겠다 선택시 바로 게임 종료
