※ 프로젝트 명과 프로젝트 기능※

-프로젝트 명: 스도쿠 게임

-프로젝트 기능- 
게임 시작 및 난이도 선택,게임 진행 (숫자 입력과 유효성 검사),게임 완료 여부 확인,완성 시 추가 선택 기능,타이머 기능,게임도중 포기(종료)시 답안 보기 선택




-프로젝트 선정 이유-

과제로서 스도쿠를 선택한 이유는 GUI를 쓰지않고도 콘솔창에서 직관적으로 게임을 진행 가능하며
배웠던것들을 많이 사용해서 만들 수 있을것같아서 선정했습니다.
